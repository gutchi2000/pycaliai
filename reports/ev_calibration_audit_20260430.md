## サマリ

`diff = p_observed - p_estimate`
- diff > 0: model が **過小** 評価 (実測 > 予測) → calibrator 上方修正必要
- diff < 0: model が **過大** 評価 (実測 < 予測) → 危険、その EV 帯のベットは控える
- |diff| < 0.02: ほぼ calibration OK
