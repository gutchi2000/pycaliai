## 仮説 H1：field_spread × 市場歪み

**タイトル:** 混戦レースにおける中位馬の市場過小評価

### 予測

field_spread を以下のように定義する：

```
field_spread = std(同一レース内スピード指数) / 出走頭数
```

field_spread が上位 25% のレース（高混戦群）において、
モデルがレース内 3〜5 番手に予測した馬の

**実際の複勝率 > 市場内包複勝率（= 1/複勝オッズ × 1.25）**

という関係が成立する。
かつ、この関係は低混戦群（下位 25%）では成立しない。

### 検証指標

```
高混戦群 3〜5 番手予測馬のROI > 低混戦群 3〜5 番手予測馬のROI
かつ 高混戦群のROI > 1.0
```

### 理論的根拠

多頭数・実力拮抗レースでは市場参加者の認知負荷が増加し、
中位馬への評価が粗くなる（認知負荷理論）。
モデルは全馬を均等に処理するため、この条件下で相対的優位を持つ。

「混戦 = どの馬も複勝に入りやすい」という単純な事実ではなく、
「混戦で市場が穴馬を系統的に過小評価する」という歪みを予測する。

### 棄却条件

- 高混戦群 ROI < 低混戦群 ROI
- または 高混戦群 ROI < 0.90（明確なマイナス）

## 仮説 H2：class_gap × 降級効果

**タイトル:** 1段降級における市場の過小評価

### 予測

class_gap を以下のように定義する：

```
class_gap = 前走クラス番号 - 現クラス番号
（昇級 = 負値、同クラス = 0、1段降級 = 1、2段降級 = 2）
```

**「1段降級（class_gap=1）の馬のROIは、
　 同クラス出走（class_gap=0）の馬のROIを上回る」**

2段降級（class_gap=2）の挙動については事前に断言しない。
（逆U字型・単調増加・フラットのいずれも理論的に排除できないため）

### 検証指標

```
1段降級ROI - 同クラスROI > +5%
（複勝ROI、◎予測馬対象、2024年7〜12月）
```

### 理論的根拠

直近不振の記憶が市場参加者の評価を歪める（recency bias）。
1段降級馬は前クラスで戦えた実力を持ちながら、
直近の負けだけを見て過小評価される。
この構造的な歪みは「1段降級 > 同クラス」として現れる。

なお、逆U字型仮説（2段降級 < 1段降級）を当初検討したが、
「市場が2段降級を察知して補正する」という推論は
recency bias の機序から演繹できず、憶測にとどまると判断。
仮説を「1段降級 > 同クラス」の1点に絞ることで棄却条件を明確化した。

### 棄却条件

```
1段降級ROI <= 同クラスROI
（差がゼロ以下なら、class_gapの追加に理論的根拠がない）
```

### 特徴量設計への含意

2段降級の形状を断言しない以上、class_gap は **連続値** として実装する。
LGBMが形状（逆U字型・単調・閾値型）を自動発見する。
実装後に部分依存プロット（PDP）で実際の形状を確認し、
H2の棄却判断とは独立して特徴量採用を評価する。

検証日: 2026-03-25
使用データ: top1_features.csv + master field_spread計算（前走RPCI std/n_horses）

| 期間 | 低spread ROI | 高spread ROI | 差 | n(高) |
|------|-------------|-------------|-----|-------|
| Valid 2023（参考） | 66.2% | 59.3% | -6.9% | 758 |
| Test H1 2024前半 | 64.6% | 68.4% | +3.7% | 413 |
| Test H2 2024後半（独立） | 64.5% | 65.1% | +0.7% | 343 |

判定: **棄却（パターンB、保留条件非該当）**
- 独立検証（H2）での差は+0.7%。保留条件（差≥+10% かつ n≥200）未達。
- 5分位分析でも単調な傾向なし（Q1:66.6%, Q2:62.3%, Q3:66.0%, Q4:61.8%, Q5:62.9%）。
- Valid 2023ではむしろ逆転（-6.9%）。

コメント:
  「混戦レースで市場が穴馬を過小評価する」という仮説は棄却。

  追加確認（2026-03-25）：高混戦 vs 低混戦のオッズ比較
    Test H2: 高spread群 単勝オッズ平均3.72 vs 低spread群5.14（差-1.41）
    複勝率差: +15.7%（高混戦の方が当たりやすい）
    ROI差:    +0.7%（ほぼゼロ）

  結論: 市場は「混戦 = 荒れやすい」をすでに正確に織り込んでいる。
  複勝率の上昇とオッズの低下がほぼ完全に相殺される。
  Valid 2023の-6.9%も「オッズが下がりすぎた期間」として説明可能。

  認知負荷仮説は「市場が混戦を正確に評価できている」という形で棄却。
  真のfield_spreadで再試行しても同様の結果になる可能性が高い。
  field_spread特徴量の追加は見送り。

## ルール廃止記録

### 重×16頭除外ルール（2026-03-25 削除）

**元のルール:**
```
baba == "重" and n_horses >= 16 → スキップ（ROI=50.5%）
```

**設定根拠（当時）:** valid 2023 単年 n=130行 ROI=55.2%

**廃止根拠:**

| 年 | n | ROI |
|----|---|-----|
| 2023（ルール根拠） | 130行 | 55.2% |
| 2024 | 142行 | 97.4% |
| 2025 | 104行 | 65.7% |
| **3年合計** | **376行** | **73.1%** |

- n=130（単年）で作成したルールを n=376 の3年観察が否定
- 3年合計ROI 73.1%は「除外すべき地獄セグメント（<60%）」の閾値を超えている
- 2024の逆転（97.4%）は n=142 で無視できる規模ではない
- 元ルールのサンプルが過少であり、ルール自体の信頼度が不十分だったと判断

**再検証条件:** n=500以上蓄積時（現在n=376）に同条件での3年以上のROI推移を確認する。
再検証時の判断基準: ROI<65%（3年平均）かつ n≥500 → 復活を検討。

### R5（EV≥3.0除外ルール）監視記録 + 仮説H3（2026-03-25 登録）

**現状（2026-03-25時点）:**
- 元ルール根拠: valid 2023 単年 ROI=59.1%（※単勝オッズ/3 近似値、実際は71.9%）
- 2年合計ROI: 約80%（全体84.8%より低い）→ 除外の方向性は維持

**観察（事後登録禁止のため仮説として登録）:**

| EV帯 | n | ROI |
|------|---|-----|
| 2.5-3.0 | 690行 | 91.7% |
| 3.0-4.0 | 661行 | 71.0% |
| 4.0+    | 664行 | 90.4% |

EV帯ROIがU字型（3.0-4.0が低く、4.0+が高い）を示している。
**ただし本観察は今日のデータを見た後の発見であり、閾値調整の根拠には使用しない。**

**仮説H3:「EV除外の問題はEV3.0-4.0帯であり、EV4.0以上は除外不要」**

- 理論的根拠（事前）: EV4.0+は市場オッズとモデル確率の乖離が最大で、
  モデルが正しければ高リターンが期待できる。
  EV3.0-4.0は「過大評価」と「真の好機」が混在する中途半端な帯の可能性がある。
- **検証条件: EV3.0-4.0 および EV4.0+ それぞれ n≥500 蓄積後**
- 事前決定ルール:
  - EV4.0+ ROI ≥ 85% かつ EV3.0-4.0 ROI < 75% → 閾値を4.0に変更
  - 上記以外 → 現行閾値3.0を維持
- 現在のEV4.0+: n=664行（2023-2024）→ 2025蓄積でn=500超を確認してから検証
